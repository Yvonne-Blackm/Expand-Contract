# Expand-Contract
Yvonne Blackmore
01.03.26

Instructions
Click the mouse to switch between scenes:
    1. Initial circle
    2. Expanding universe: an expanding central circle of particles, and new particle circles starting at the mouse position
    3. Dark matter: particles are attracted to an invisible object
From an initial static state/universe, an expansion takes place, followed by a contraction as particles are attracted to an invisible object representing dark matter.
